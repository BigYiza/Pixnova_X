# 数据模型

## Account

用户、权益、邀请、AB 分组的本地聚合模型。详见 `03_用户体系.md`。

## HomePageDataSource

字段：

- `cards`
- `banners`
- `pops`
- `kingkongPlaces`
- `newUserAsk`
- 本地 `tabs`

## AllDataSource

字段：

- `cards`

用途：

- 全量模板池。
- 各 AI 模块根据 AIType 或 show_position 查找模板。

## UploadModel

一次 AI 制作的运行时模型：

- 输入素材。
- 选中模板。
- prompt。
- external_args。
- combine_configs。
- 上传/生成状态。
- 错误。
- 结果。

## DiamondsExpenseModel

高级参数模型：

- `parameters`
- `restricts`

最终影响：

- UI 参数选项。
- 参数联动。
- 钻石价格。
- `external_args`。

## AITaskItem

历史任务模型：

- `create_time`
- `filter`
- `msg`
- `result`
- `state`
- `task_id`

## RewardTask

奖励中心任务：

- `diamonds`
- `id`
- `name`
- `description`
- `sub_type`
- `task_type`
- `value`
- `completed`
