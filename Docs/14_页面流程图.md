# 页面流程图

## 启动流程

```text
PixnovaApp
  -> AppDelegate
    -> LaunchConfig 初始化
    -> 注册推送
    -> 创建数据库
  -> ContentView
    -> LaunchScreenLogoView
    -> 首次启动 GuideIntroduceView
    -> HomePage
```

## 首页流程

```text
HomePage
  -> HomePageContrainer
    -> Home tab
    -> AI Video tab
    -> AI Filters tab
    -> AI Photo tab
    -> Me tab
  -> 点击 banner/card/kingkong
    -> 根据 action 或 AIType 跳转
```

## 模板点击分流

```text
FilterItem
  -> avatar: AvatarDesignIntroPage
  -> oneImageSet: OneBigImagePage
  -> baby: BabyGuideView
  -> video: VideoCreatePage
  -> imageToVideo/textToVideo/multiImageToVideo: CustomVideoEditPromptView
  -> videoResolution: VideoResolutionGuideView
  -> imageToImage/textToImage: CustomImageEditPromptView
  -> 其他: FilterPage
```

## Filter 制作

```text
FilterPage
  -> choose photo
  -> 钻石校验
  -> upload state
  -> UploadPage
  -> display state
  -> save/share/retry/no watermark/use related video
```

## Custom Video

```text
CustomVideoEditPromptView
  -> 选择 Text/Image/MultiImage
  -> 选择图片或输入 prompt
  -> 高级参数
  -> 钻石校验
  -> UploadPage
  -> VideoDisplayPage
```

## Custom Image

```text
CustomImageEditPromptView
  -> 选择 TextToImage/ImageToImage
  -> 输入 prompt/多图
  -> 高级参数
  -> 钻石校验
  -> UploadPage
  -> ImageDisplayPage
```

## 支付

```text
MemberShipView / DiamondPurchaseView
  -> load SKProduct
  -> create order
  -> StoreKit payment
  -> success_notify
  -> order status
  -> refresh membership state
```

## 个人中心

```text
MineView
  -> query_my_tasks
  -> MineDisplayView
    -> download/share/delete/use template
```

## 推送

```text
FCM/APNs
  -> AppDelegate didReceive
  -> NotificationCenter push_message
  -> HomePage processPusMessage
```
