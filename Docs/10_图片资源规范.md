# 图片资源规范

## 资源来源

当前工程资源分布：

- `Assets.xcassets`：图片、图标、背景。
- `Resources`：APNG、GIF、JSON、MOV/MP4、字体。
- `Resources/GuideIntroduce`：引导页视频。
- Firebase/OSS/CDN：后端配置的封面、模板、生成结果。

## 主要资源类型

| 类型 | 用途 |
| --- | --- |
| PNG/JPG | 图标、背景、模板封面、按钮 |
| GIF | 会员/钻石旋转动画 |
| APNG | 生成 loading、手势引导 |
| JSON | Lottie loading |
| MOV/MP4 | 引导视频、会员视频、视频增强示例 |
| TTF | Rubik 字体 |

## 本地生成/保存资源

| 路径 | 用途 |
| --- | --- |
| Firebase Storage `iOS/uploadImage/yyyyMMdd/UUID` | 上传素材 |
| OSS `file_url` | 上传素材 |
| Documents `origin_images/{localId}` | 作品原图 |
| Documents 视频缓存 | `VideoDownloadManager` 下载视频 |
| Documents `pixnova.sqlite` | 作品本地关联 |
| Documents `analytics.sqlite` | 自研埋点缓存 |

## 水印规则

- 非 VIP 展示结果时叠加 `filter_watch_mark`。
- 非 VIP 保存图片时使用 `FilterMardImageView` 渲染带水印图片。
- VIP 保存原始生成图。

## 模板封面字段

| 字段 | 用途 |
| --- | --- |
| `cover` | 常规封面 |
| `cover1` | 备用封面 |
| `operations_cover` | 视频/操作封面 |
| `relation_card_media_desc` | 结果页挂载视频预览 |

## 新工程建议

- 不复用老产品品牌、启动页、引导视频、会员页素材。
- 图标和按钮资源按功能命名，不按旧页面命名。
- 生成结果、用户上传图片、缓存视频需要明确缓存和删除策略。
- 隐私政策需说明用户素材上传到云端用于 AI 生成。
