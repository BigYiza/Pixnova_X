# 业务规则

## 权益规则

- VIP 可用全部 VIP 模板。
- 非 VIP 不能使用 cutout。
- 非 VIP 生成图展示和保存时加水印。
- VIP 可跳过等待进入后台生成。
- 钻石不足时不能发起钻石消耗型生成。

## 图片输入规则

- `image_count == 1`：需要单张图片。
- `image_count == 2`：需要两张图片。
- 多图生视频/图生图：数量来自 `combine_configs.function_configs.input_require.image_count`。
- `max_input_count` 控制最大可上传数量。

## 视频输入规则

- 视频分辨率提升必须有本地视频 URL。
- 上传时根据文件头识别视频格式。
- 结果下载到本地后播放和分享。

## 生成规则

- `filter_id` 必须大于 0。
- 创建任务成功必须返回 `data.id`。
- 轮询成功必须返回 `data.aigc_url`。
- 默认轮询间隔 0.5 秒。
- 超时由 `wait_seconds` 控制，默认 120 秒。

## 钻石价格规则

- 基础价格：`filterItem.diamonds ?? 10`。
- 高级参数每个 selected value 的 `times` 作为倍率。
- 视频增强可额外按时长倍率计算。
- 最终 `external_args` 传所选参数 key/value。

## 任务规则

- 用户取消制作：若已有 taskId，调用删除任务接口。
- 后台任务：保存 taskId 到本地。
- 制作成功：保存 taskId、filterId、generateUrl、原图映射。
- 作品详情删除：调用同一个删除任务接口。

## 邀请规则

- 注册 3 天内可填写邀请码。
- 已有 `invited_code` 则不可再填。
- 邀请奖励变化通过本地差值判断是否弹窗。

## 奖励任务规则

- 服务端 `completed == true` 代表已完成/已领取。
- 本地完成但未领取时展示 Claim。
- 领取成功后刷新任务和会员状态。

## 评分规则

- 保存图片/视频成功后计数。
- 第 0 次和第 2 次保存成功触发系统评分。
- 新产品建议重新设计触发规则，避免过早或频繁打扰。

## 审核与配置规则

- `is_reviewing` 当前会影响促销和会员 AB 展示。
- 新产品应把该状态用于合规降级，而不是内容切换。
- RemoteConfig 默认值必须合理，避免网络失败导致关键流程不可用。
