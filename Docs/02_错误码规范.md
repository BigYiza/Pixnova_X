# 错误码规范

## 统一响应规则

普通业务接口成功条件：

1. HTTP 状态码 2xx。
2. JSON 能解析为字典。
3. 存在 `code: Int`。
4. `code == 0`。

失败文案读取优先级：

1. `description`
2. `desc`
3. `msg`
4. 本地化 `request fail`

## 本地错误类型

| 类型 | 含义 | 文案 |
| --- | --- | --- |
| `requestError` | 网络请求失败 | 当前网络连接断开，请稍后再试 |
| `mappingError` | HTTP/JSON/模型解析失败 | 当前网络连接断开，请稍后再试 |
| `unzipError` | 预留解压错误 | 文件解压失败 |
| `serverError` | 服务端业务失败 | 服务端 msg |

## 已知业务 code

| code | 含义 | 当前处理 |
| --- | --- | --- |
| `0` | 成功 | 正常解析 |
| `-100` | token 失效 | 调 `refreshToken()`，返回 `Token Failed` |
| `-101` | 会员/VIP 不足 | 打开会员页，Gemini 触发 memberBlock |
| `-102` | 视频次数不足 | 清空 `videoTimes` |
| `-103` | 钻石不足/需购买 | 打开钻石购买或刷新权益 |
| `-1` | 本地通用失败 | 参数错误、未知错误、下载失败 |
| `99` | Firebase 上传 URL 异常 | upload Image Failed |

## 制作链路错误

| 阶段 | 失败文案 |
| --- | --- |
| 参数检查 | `Parameter error` |
| 上传图片 | `Failed to upload image` |
| 上传视频 | `Failed to upload video` |
| 创建任务无 id | `Failed to obtain taskId` |
| 查询结果失败 | `Failed to obtain url` |
| 下载结果失败 | `Failed to download image` |
| 超时 | `Generate timeout` |
| 未知 | `Unknown error` |

## 支付错误

StoreKit 错误在会员和钻石购买 ViewModel 中映射：

- `paymentCancelled`：Purchase Cancelled
- `storeProductNotAvailable`：Product Not Available
- 其他使用本地化错误文案或 `error.localizedDescription`

支付域名接口由业务层判断，不走普通 `code` 失败分支。
