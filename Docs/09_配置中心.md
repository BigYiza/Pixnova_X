# 配置中心

## 配置来源

当前工程配置来源有四类：

1. 编译环境常量。
2. Firebase RemoteConfig。
3. 后端业务配置接口。
4. UserDefaults 本地开关。

## 编译环境

`LaunchConfig.swift` 根据宏控制：

- `DEBUG`
- `ADHOC`
- `RELEASE`

控制内容：

- 主业务域名。
- 自研埋点域名。
- Firebase plist 名称。

## Firebase RemoteConfig

初始化：

- `minimumFetchInterval = 0`
- App 启动 fetchAndActivate

已使用 key：

| Key | 用途 |
| --- | --- |
| `BabyGeneraingConfiguration` | Baby prompt 模板 |
| `avatarConfiguration` | Avatar 风格和 prompt 配置 |
| `membershipViewABTest` | 会员页 AB 旧逻辑 |
| `memberDiscountTimeLimit` | 限时优惠倒计时分钟数 |

## 后端配置

| 接口 | 配置 |
| --- | --- |
| `/openapi/query_status_by_ver` | `is_reviewing` |
| `/api/query_user_group_by_pos` | 会员页分组 |
| `/api/query_operations_data` | 首页运营弹窗、Ask、tabs |
| `/api/query_cards` | 全量模板 |
| `/api/query_custom_parameter` | 高级参数 |

## UserDefaults 开关

| Key | 说明 |
| --- | --- |
| `isFirstLaunch` | 是否首次启动 |
| `isFirstLaunchAsk` | 是否首次首页 Ask |
| `currentUserAccount` | 当前账号 |
| `cached_all_data` | AllData 缓存 |
| `cached_homepage_data_type_{type}` | 首页缓存 |
| `AppHasPreDownloadAskResource` | Ask 资源是否已预加载 |
| `kBackgroundTasksKey` | 后台生成任务 |
| `ignoreInviteInfoDialog` | 是否忽略邀请弹窗 |
| `hasClickedRelationVideoBubble` | 结果页挂载视频气泡 |
| `v_1.1.1_genera_noti` | 生成页提示 |
| `v2.15_image_to_video_guide` | 多图生视频引导 |
| `2_9_0_good_review_count` | 评分触发计数 |
| `reward_task_completed_{id}` | 奖励任务本地完成 |

## 新工程建议

- 配置项建立清单和默认值。
- RemoteConfig key 重新命名为新产品语义。
- 审核状态只能用于合规展示降级，不应作为规避机制。
- 本地开关需要版本化，避免老状态污染新产品。
