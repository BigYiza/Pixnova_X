# AI 工作流

## 统一生成管线

绝大部分 AI 功能复用：

```text
页面输入
  -> UploadModel
  -> 上传素材到 OSS/Firebase
  -> /api/image_generation 创建任务
  -> /api/query_result 轮询
  -> 下载结果资源
  -> 展示/保存/刷新权益
```

## 创建任务参数

基础参数：

```json
{
  "urls": [],
  "filter_id": 0
}
```

可选参数：

| 参数 | 来源 |
| --- | --- |
| `prompt` | 用户输入或模板 prompt |
| `negative_prompt` | 用户改写模板 prompt 时置空 |
| `external_args` | 高级参数 |
| `combine_configs` | 多图组合配置 |

## 高级参数

后端字段：

- `custom_parameter.parameters`
- `custom_parameter.restricts`

本地逻辑：

- 参数之间有联动约束。
- 默认选中值来自 `selected`。
- 价格 = 基础钻石 * 参数倍数。
- 最终请求放入 `external_args`。

## 后台任务

支持后台的任务：

- AI Video
- Custom Video
- Custom Image
- Video Resolution

流程：

- 任务创建后才能跳过等待。
- VIP 可进入后台任务。
- taskId 保存到 `kBackgroundTasksKey`。
- 通过推送或个人中心查看。

## Gemini 例外

Gemini 不走统一生成管线：

- 上传图片仍可用 Firebase。
- 请求 `/api/chat_generation_content`。
- `Accept: text/event-stream`。
- 流式 JSON 里 `type == text/images`。
- `code == -101` 打开会员。
